RepairRx - Professional Device Diagnostic Tool

Quick Start

RepairRx is your all-in-one solution for professional device diagnostics and repair workflow management.

Overview

RepairRx streamlines the device repair process with an intuitive interface designed specifically for repair technicians and service centers.

Typical Check-in Scenario

  1. Customer arrives with device issues
  2. Technician launches RepairRx
  3. Selects device type from dropdown (iPhone, Samsung, Google Pixel, Other Android)
  4. Runs preliminary diagnostics
  5. Generates diagnostic report for customer
  6. Creates repair ticket with findings

Key Features

  • Hardware component testing
  • Battery health analysis
  • Boot diagnostics
  • Data backup utilities
  • Recovery mode tools
  • Comprehensive diagnostic reports

System Requirements

  • Python 3.8 or higher
  • Tkinter (included with Python)
  • Pillow (PIL)
  • Operating System: Windows 10/11, macOS 10.15+, or Linux
  • Minimum 4GB RAM
  • 100MB free disk space
  • USB ports for device connection
  • Internet connection for updates

Installation Instructions

1. Install Python

Windows:

Download from https://www.python.org/downloads/

macOS:

brew install python

Linux:

sudo apt-get update
sudo apt-get install python3

2. Install Required Packages

pip install pillow

3. Clone and Run

git clone https://github.com/yourusername/repairrx
cd repairrx
python DiagnoseMe.py
