# RepairRx - Professional Device Diagnostic Tool

## Overview
RepairRx is a comprehensive diagnostic tool designed for mobile device repair technicians to streamline the device check-in and testing process. It provides a systematic approach to device diagnostics with an intuitive graphical interface.

## Use Cases
### Typical Check-in Scenario
1. Customer arrives with device issues
2. Technician launches RepairRx
3. Selects device type from dropdown
4. Runs preliminary diagnostics
5. Generates diagnostic report for customer
6. Creates repair ticket with findings

## Features
- Hardware component testing
- Battery health analysis
- Boot diagnostics
- Data backup utilities
- Recovery mode tools
- Comprehensive diagnostic reports

## Requirements
- Python 3.8 or higher
- Tkinter (included with Python)
- Pillow (PIL)
- Operating System: Windows 10/11, macOS 10.15+, or Linux
